Proyecto Javier Palacio

PROYECTO FIN DE CURSO DE MULTIPLAFORMA 2015-2016.

Se realiza una app con Android Studio, que una vez logeado el usuario obtenga el número de pasos dados por él en el día y que estan registrados en Google Fit.

Una vez tenemos el usuario, la fecha del dia y el numero de pasos, los subimos a la nube en la misma app.

Se desarrolla una página web que lea los datos de la nube y que visualizarel número de pasos dados por cada usuario, durante distintos días.

También se visualizan unos gráficos con librerías de terceros.

Para que de esta manera, se pueda llevar un pequeño control del ejercicio que desarrolla a lo largo del día cada usuario.

Tecnologias usadas.

Android Studio

Android Studio es un entorno de desarrollo integrado para la plataforma Android, la primera versión estable fue publicada en diciembre de 2014. Es publicado de forma gratuita y está disponible para las plataformas Microsoft Windows, Mac OS X y GNU/Linux.

Parse .com

Parse produce herramientas de back-end para los desarrolladores móviles que ayudan a los desarrolladores móviles almacenar datos en la nube, la gestión de identidad log-ins, manejar notificaciones push y ejecutar código personalizado en la nube.

Plot.ly

Es una herramientas de análisis y visualización de datos. Proporciona gráficos en línea, análisis y Estadísticas herramientas para las personas y la colaboración, así como las bibliotecas gráficas científicas para distintos lenguajes de programacion.
